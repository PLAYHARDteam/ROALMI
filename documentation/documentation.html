How to Place it On Server?

To place your template on server, Login into your hosting C panel OR connect FTP (file transfer protocol). Go to a Root directory > Upload .zip OR theme folder to that directory. If you are going to upload .zip folder than you have to extract that folder in the directory.

Once done, Visit the URL of your site.

Files Structure

  • /css (Template CSS)
  • /fonts (Font Awesome, etc..)
  • /images (All Images)
  • /js (Template JS)
  • /video (All Videos)

HTML Structure

The template is based on Bootstrap 3 Framework - http://getbootstrap.com

Bootstrap includes a responsive, mobile first fluid grid system that appropriately scales up to 12 columns as the device or viewport size increases.

<div class="row">
              <div class="col-md-12">
                    Level 1 column
                    <div class="row">
                             <div class="col-md-6">Level 2</div>
                             <div class="col-md-6">Level 2</div>
                    </div>
              </div>
 </div>

If you need more information about Bootstrap and Grid system, please visit this site: http://getbootstrap.com/css/#grid

CSS Structure

Following are the CSS files we're using in t-designfest template:

  • style.css (Add all customizations into this file)
  • responsive.css (Add all customizations for mobile layout into this file)

Fonts

To change the inserted font, please take a look in the head part of the website and you will find this tag:

@import url('https://fonts.googleapis.com/css?family=Open+Sans:300,400,600,700,800');

If you want to change the font & font style, go to http://www.google.com/webfonts choose a font you want to add and use the generated code for particular font.

Icons

Font Awesome is a @font-face iconset by which you can change size and color of the icons using CSS. If you need more information, please visit this site: http://fontawesome.io/

ElegantIcons is also used in this template. If you need more information about ElegantIcons, please visit this site: https://www.elegantthemes.com/blog/resources/elegant-icon-font

LightBox

The lightbox is operated by Magnific Popup script. Here's an example of how to use it:

 <a href="images/corporate-portfolio-item12.jpg" data-image-popup="">
    <img src="images/corporate-portfolio-item12.jpg" alt="portfolio-image"></a>

For more information check the documentation.

http://dimsemenov.com/plugins/magnific-popup/

Video Background

You can customize the video background section by making required changes in the following code:

<video autoplay="" muted="" loop="" class="html-video" preload="auto" 
poster="images/video-bg1.jpg">
            <source type="video/mp4" src="video/vid-island.mp4">
            <source type="video/ogg" src="video/vid-island.ogg">

    </video>

Google Maps

As per Google recent announcement, usage of the Google Maps APIs now requires a key. If you are using the Google Maps API it will require a key going forward. To get the API key, you must follow

1) Get an API key: https://developers.google.com/maps/documentation/javascript/get-api-key

2) Add the API key in the HTML (Script) here,

<script src="https://maps.googleapis.com/maps/api/js?
               key=AIzaSyC9HraXXqhlEEWIDVBMinxQJVabqPNV9uo"></script>
            <script src="js/map-custom.js">Level 2</script>

Features

  • Mobile Friendly - For mobile friendly/responsive layout, no special actions required, all pages included in this template are mobile-friendly. You don't have to create a special code or css for mobile version of your t-designfest, it will adjust automagically.
  • Well Coded Files - This template is entirely built with HTML5, CSS3, jQuery and Bootstrap framework. It include 120+ .html files, 15+ .css files, 20+ .js files which all have well-structured code with clean design.
  • Easy And Simple - The customization of the template is really easy and quick. It cut down the development time of your website and saves the coding hours as well as your time. You just have to put your content and links in the particular section you want - no technical skills are required to make a changes.
  • Unique Styles - T-designfest inlcudes a number of homepage layouts with an eCommerce option also from which you can choose the best match for your business. It comes with full-screen banner, bootstrap carousel, image gallery with lightbox, parallax scrolling, video backgrounds, megamenu, sticky header for onepage layout and more.
  • Elements - The template inlcudes web-elements like 10+ page titles style, accordions, alert message, progress bar, buttons, grid, icons et-line, icons font awesome, tabs, typography etc. Portfolio also has different layouts including grid, grid-gutter, wide, wide-gutter, grid & wide with title, masonry & single project page.
